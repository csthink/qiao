## 实现摘要
- **trait**:`get -> Secret` / `set` / `exists -> bool`,全 `anyhow::Result`。`Secret = Zeroizing<String>`,离开作用域即清零。
- **keychain**:`keyring::Entry::new(SERVICE, &account)`。`exists()` 即使必须读出明文(keyring 无纯存在性 API),也立刻 `Zeroizing` 接管并 drop,只回 bool。`NoEntry` 映射为 `qiao key set ...` 指引。
- **bw**:locator `item/<名>`/`id/<id>` → `bw get password <值>`;**key 只经 stdout**(不进 argv)。失败只读 stderr(不含明文),`classify` 分 NotLoggedIn/Locked/NotFound/Other,`actionable` 给 `bw login`/`bw unlock` 等可操作消息。set 在 v1 返回"手动创建"指引(不把 key 经 argv 喂 bw create)。**绝不引入 bws**。
- **env**:`get` 读变量,`exists` 用 `var_os().is_some()` 不读出明文;`set` 拒绝(红线:key 不进持久化 env)。
